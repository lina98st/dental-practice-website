# Dental Practice Website

Modern website for a medical or dental practice.  
Built with React and Vite as a frontend only project without a backend.

## Tech Stack
- React
- Vite
- JavaScript
- HTML
- CSS

## Project Goal
The goal of this project is to create a clear, trustworthy and modern practice website with the following sections:
- Home
- Services
- Team
- Contact
- Opening Hours

## Project Setup

### Requirements
- Node.js version 18 or higher recommended
- npm

### Installation
```bash
npm install

 
Start Development Server
npm run dev

The website will be available locally, usually at
http://localhost:5173

Project Structure
src/
 components/   Reusable UI components
 pages/        Page components
 assets/       Images and static files
