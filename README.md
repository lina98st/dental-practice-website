# Dental Care Center Website

Modern, responsive website for a dental practice.  
Built with React and Vite as a frontend-only project.

## Tech Stack
- React
- Vite
- JavaScript
- HTML5
- CSS3

## Features
- Fully responsive design (mobile, tablet, desktop)
- Modern, soft color scheme with gradients
- Auto-playing image carousel
- Smooth scroll navigation
- Clean and accessible UI
- Fast performance with Vite

## Project Sections
- **Hero** - Welcome section with call-to-action
- **About** - Information about the practice
- **Image Carousel** - Showcase of the clinic
- **Services** - Overview of dental services offered
- **Opening Hours** - Weekly schedule
- **Contact Info** - Address, phone, email, emergency contact

## Project Setup

### Requirements
- Node.js version 18 or higher
- npm or yarn

### Installation
```bash
npm install
```

### Start Development Server
```bash
npm run dev
```

The website will be available locally at `http://localhost:5173`

### Build for Production
```bash
npm run build
```

### Preview Production Build
```bash
npm run preview
```
